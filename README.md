# 🌵 Xeriscaping Landing Pages – Multi Demo UIs 

A premium multi concept landing page system for a modern xeriscaping and landscape design brand.
Built to showcase **luxury outdoor design, sustainability, and high conversion UI experiences** across multiple unique visual directions.

---

## ✨ Project Overview

This project is a collection of **three distinct landing page concepts** for a xeriscaping company.

Each page represents a different creative direction:

* **Page 1:** Clean, modern, conversion focused design system
* **Page 2:** Editorial, cinematic storytelling experience
* **Page 3:** Experimental, immersive, motion rich UI concept

Together, they demonstrate how a single brand can evolve across multiple premium digital identities.

---

## 🌿 Key Features

* Multiple unique landing page designs
* Fully custom UI for each page
* Premium xeriscaping themed visuals
* High quality hero sections with strong first impression focus
* Smooth animations and micro interactions
* Conversion focused layouts
* Scroll based storytelling experience
* Component based architecture
* Isolated pages (no shared UI conflicts)
